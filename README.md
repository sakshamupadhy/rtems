# rtems
This project is the real-time Executive for Multiprocessor system (RTEMS).
The current version of the software can be found in the VERSION file.


you can access the documenation manuals in the docs/ directory, with builds available online at
https://rtems.org/onlinedocs/doc-current/share/rtems/html/ and released version at
http://www.rtems.org/onlinedocs/releases/ for guidance on building, installation, and 
usage of RTEMS
refer to INSTALL file for additional setup instruction


for community resources, tutorials, and shared knowledge, visit the RTEMS wiki at https://devel.rtems.org

RTEMS API documentation (Doxygen) is available at http://www.rtems.org/onlinedocs/doxygen/cpukit/html

The RTEMS project maintains mailing lists for communication and discussions:

.For general usage-related questons, use the rtems-users mailing list:
https://lists.rtems.org/mailman/listinfo/devel

to explore exsisting issues or report a new one, visit:
https://devel.rtems.org/
